# clone repository
```bash
   git clone https://github.com/PluemDontKnowToCode/MaiChaiMouseButMe
   cd MaiChaiMouseButMe
```

# build venv

```bash
   python -m venv myenv
```

# เปิดใช้งาน venv
# Windows
```bash
   myenv\Scripts\activate
```
# Linux/Mac
```bash
   source myenv/bin/activate
```
